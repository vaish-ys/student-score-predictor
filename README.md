# Student Score Prediction

## 📌 Project Description
This project predicts a student's final score based on:
- Hours studied
- Sleep hours
- Previous score

## 🛠️ Technologies Used
- Python
- pandas
- scikit-learn
- matplotlib

## 🚀 How to Run
1. Open Jupyter Notebook
2. Run all cells
3. See prediction output

## 📊 Output
The model predicts student performance using Linear Regression.
